# L8nes 🛹

> Skate spots. Your city. Your level.

LA-first skater app for discovering spots, sharing clips, and connecting with local skaters.

## Stack
- **React Native** + Expo (Expo Router)
- **Supabase** — auth, database, storage, realtime
- **Mapbox** — spot maps
- **TanStack Query** — server state
- **Zustand** — local state

## Screens
- **Feed** — skill-filtered clips from nearby skaters
- **Map** — interactive spot map with live check-ins
- **Search** — find spots by neighborhood, type, skill level
- **Profile** — skater card, clips, favorite spots
- **Spot Detail** — full spot info, clips, check-in
- **Skater Profile** — follow, view clips & spots added

## Getting Started

```bash
npm install
npx expo start
```

Scan the QR code with **Expo Go** on your phone.
